# Calculadora_notas_moedas

Projeto desenvolvido como atividade da disciplina de Programação para Internet, com o objetivo de aplicar conceitos de HTML, CSS e JavaScript na construção de uma interface interativa e lógica de cálculo.

# Descrição

A aplicação lê um valor monetário (com duas casas decimais) e calcula o menor número possível de notas e moedas necessárias para representar esse valor.

As cédulas e moedas consideradas são:

-Notas

R$ 100.00
R$ 50.00
R$ 20.00
R$ 10.00
R$ 5.00
R$ 2.00

-Moedas

R$ 1.00
R$ 0.50
R$ 0.25
R$ 0.10
R$ 0.05
R$ 0.01

# Tecnologias utilizadas

HTML5 → estrutura da página
CSS3 → estilização (layout estilo app com tema roxo/preto)
JavaScript → lógica de cálculo e manipulação do DOM

# Como funciona
O usuário insere um valor decimal (ex: 576.73)
O valor é convertido para centavos para evitar erros com ponto flutuante
O sistema calcula:
quantidade de notas
quantidade de moedas
O resultado é exibido na tela, organizado linha por linha

📥 Exemplo de Entrada
576.73 
📤 Exemplo de Saída

NOTAS:
5 nota(s) de R$ 100.00
1 nota(s) de R$ 50.00
1 nota(s) de R$ 20.00
0 nota(s) de R$ 10.00
1 nota(s) de R$ 5.00
0 nota(s) de R$ 2.00

MOEDAS:
1 moeda(s) de R$ 1.00
1 moeda(s) de R$ 0.50
0 moeda(s) de R$ 0.25
2 moeda(s) de R$ 0.10
0 moeda(s) de R$ 0.05
3 moeda(s) de R$ 0.01
 
# Este projeto teve como objetivo:

Praticar manipulação de DOM com JavaScript
Trabalhar com eventos (addEventListener)
Aplicar conceitos de lógica de programação
Separar responsabilidades (HTML, CSS e JS)
Criar uma interface amigável para o usuário.
